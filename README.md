# Go, Rust, TS Playground

## Initialization

```bash
go mod init git
yarn init -y
cargo init
```

## Check installed versions

```bash
go version
rustc --version
yarn tsc --version
```

## Typescript

```bash
yarn add ts-node @types/node typescript
```

```
yarn tsc --init
```

```
yarn ts-node src/test.ts
```

## C++

```bash
g++ -o test ./src/test.cpp

./test
```

## Rust

```bash
cargo run
```
